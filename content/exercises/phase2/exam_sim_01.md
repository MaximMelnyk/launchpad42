# Exam Simulation 1 (4h)

## Завдання

Це твоя **перша симуляція іспиту Piscine**. На реальній Piscine іспити проходять 3 рази по 4 години (п'ятниці) та один фінальний 8 годин. Ця вправа -- процедурний гайд та перелік завдань для домашньої симуляції.

**Мета:** перевірити свої знання в умовах, максимально наближених до реальних. Навіть якщо отримаєш 0 балів -- це нормально для першого разу. Головне -- відчути формат та зрозуміти, що потрібно покращити.

## Підготовка (перед стартом)

### Налаштування середовища

1. **Закрий все зайве:** браузер, месенджери, музику, YouTube
2. **Відключи інтернет** (Wi-Fi off або Flight Mode на роутері)
3. **Тільки дозволені інструменти:**
   - Термінал (bash)
   - Текстовий редактор: **Vim** або **Emacs** (НЕ VS Code!)
   - Компілятор: `gcc -Wall -Wextra -Werror`
   - Man сторінки: `man 2 write`, `man 3 strlen` тощо
4. **Встанови таймер:** рівно 4 години (240 хвилин)
5. **Створи робочу директорію:**

```bash
mkdir -p ~/exam_sim_01
cd ~/exam_sim_01
```

### Правила

- Жодного інтернету, жодних нотаток, жодних шпаргалок
- Тільки `man` сторінки дозволені для довідки
- Кожне завдання -- окрема директорія (`ex00/`, `ex01/`, тощо)
- Компілюй з `-Wall -Wextra -Werror`
- Norminette обов'язкова
- Тільки `write()` -- жодного `printf`

## Завдання іспиту

### Рівень 0 (Level 0) -- вибери одне

| # | Завдання | Опис | Очікуваний час |
|---|----------|------|----------------|
| A | ft_putchar | Вивести один символ | 2 хв |
| B | ft_print_numbers | Вивести "0123456789" | 3 хв |
| C | ft_countdown | Вивести "9876543210" | 3 хв |

Якщо правильно -- переходиш на Level 1. Якщо ні -- ще одне завдання Level 0.

### Рівень 1 (Level 1) -- вибери одне

| # | Завдання | Опис | Очікуваний час |
|---|----------|------|----------------|
| A | ft_strlen | Довжина рядка | 5 хв |
| B | ft_strcpy | Копіювання рядка | 5 хв |
| C | ft_swap | Обмін двох int через pointers | 5 хв |
| D | ft_putstr | Вивести рядок | 5 хв |
| E | repeat_alpha | 'a'->1x, 'b'->2x, 'c'->3x... | 10 хв |
| F | rot_13 | ROT13 шифрування | 10 хв |

### Рівень 2 (Level 2) -- вибери одне

| # | Завдання | Опис | Очікуваний час |
|---|----------|------|----------------|
| A | ft_atoi | Рядок в число (з пробілами та знаками) | 15 хв |
| B | ft_strcmp | Порівняння двох рядків | 10 хв |
| C | ft_strdup | Дублікат рядка (malloc) | 10 хв |
| D | ft_strrev | Розвернути рядок in-place | 10 хв |
| E | ft_print_hex | Число в hex (putnbr_base 16) | 15 хв |

## Процедура виконання

### Крок 1: Обери та виконай Level 0

```bash
mkdir -p ex00
cd ex00
vim ft_putchar.c    # або інше завдання
gcc -Wall -Wextra -Werror -o test ft_putchar.c main.c
./test
```

### Крок 2: Самоперевірка

Після кожного рівня перевір вручну:
- Компілюється без помилок?
- Вивід відповідає очікуваному?
- Norminette пройдена?

### Крок 3: Перехід на наступний рівень

Якщо впевнений у відповіді -- переходь на наступний рівень. Якщо помилка -- спробуй інше завдання того ж рівня.

### Крок 4: Запис результатів

Після закінчення часу (або після завершення всіх рівнів) створи файл з результатами:

```bash
cat > ~/exam_sim_01/results.txt << 'EOF'
Exam Simulation 1 Results
Date: YYYY-MM-DD
Duration: XXX minutes (of 240)

Level 0: PASS/FAIL - [task name] - [minutes spent]
Level 1: PASS/FAIL - [task name] - [minutes spent]
Level 2: PASS/FAIL - [task name] - [minutes spent]

Total level reached: X
Notes: [what was hardest, what to review]
EOF
```

## Оцінювання

| Результат | Оцінка | Коментар |
|-----------|--------|----------|
| Level 0 | 25% | Мінімум базових знань |
| Level 1 | 50% | Базовий рівень -- нормально для першої спроби |
| Level 2 | 75% | Добрий результат! |
| Level 2 + час залишився | 100% | Відмінно! Готовий до Level 3 |
| 0 (не пройшов Level 0) | 0% | Не хвилюйся -- перший раз завжди найскладніший |

**На реальній Piscine:** перший іспит -- Level 0 часто отримують 30-50% студентів. Це нормально!

## Після іспиту

1. **Запиши результати** у `results.txt`
2. **Визнач слабкі місця:** яке завдання було найскладнішим?
3. **Повернись до вправ:** повтори ті функції, які не зміг написати
4. **Vim comfort:** якщо Vim заважав -- потренуй базові команди (`i`, `Esc`, `:wq`, `dd`, `yy`, `p`)

## Підказки

<details>
<summary>Підказка: стратегія іспиту</summary>

1. **Level 0 -- не витрачай більше 5 хвилин.** Це повинно бути автоматично.
2. **Level 1 -- обирай те, що знаєш найкраще.** ft_strlen або ft_putstr -- найшвидші.
3. **Level 2 -- ft_strcmp зазвичай найшвидший.** ft_atoi складніший, але дає більше впевненості.
4. **Якщо застряг -- перейди до іншого завдання того ж рівня.** На реальному іспиті тебе перекидає автоматично.
5. **Компілюй часто!** Не пиши 50 рядків і потім компілюй. Пиши 5-10 рядків, компілюй, тестуй.

</details>

<details>
<summary>Підказка: Vim мінімум для іспиту</summary>

Команди, які потрібно знати напам'ять:

```
i          -- insert mode (почати писати)
Esc        -- normal mode (вийти з insert)
:wq        -- зберегти та вийти
:q!        -- вийти без збереження
dd         -- видалити рядок
yy         -- копіювати рядок
p          -- вставити
/text      -- пошук "text"
:set nu    -- показати номери рядків
u          -- undo
```

</details>
