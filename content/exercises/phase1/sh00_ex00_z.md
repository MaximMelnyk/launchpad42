# Z

## Завдання

Створи файл з назвою `Z`, який при виконанні команди `cat Z` виведе рівно два символи: велику літеру `Z` та переведення рядка.

Це перша вправа модуля Shell00 у Piscine. Вона максимально проста, але вчить тебе важливому: **точність має значення**. На Piscine Moulinette (автоматичний грейдер) перевірятиме вивід побайтово. Зайвий пробіл, відсутній `\n` -- і оцінка 0.

### Вимоги

- Файл має називатися рівно `Z` (велика літера, без розширення)
- `cat Z` повинен вивести: `Z` з подальшим переведенням рядка (`\n`)
- Файл має бути рівно 2 байти (символ `Z` + `\n`)
- Здай файл у директорії `ex00/`

### Очікуваний результат

```bash
$ cat -e Z
Z$
```

Символ `$` у виводі `cat -e` позначає кінець рядка (наявність `\n`).

### Як створити файл

Є кілька способів -- обери будь-який:

```bash
# Спосіб 1: echo
echo "Z" > Z

# Спосіб 2: printf
printf "Z\n" > Z
```

## Підказки

<details>
<summary>Підказка 1</summary>

Команда `echo "Z"` автоматично додає `\n` в кінці. Тому `echo "Z" > Z` створює файл рівно з 2 байтами.

Перевір розмір файлу: `ls -l Z` або `wc -c Z`.

</details>

<details>
<summary>Підказка 2</summary>

Щоб перевірити, що файл містить саме те, що потрібно, використай:

```bash
xxd Z
```

Очікуваний вивід:
```
00000000: 5a0a                                     Z.
```

`5a` -- це ASCII-код літери `Z`, `0a` -- це `\n`.

</details>
