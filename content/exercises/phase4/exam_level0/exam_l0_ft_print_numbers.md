# ft_print_numbers

## Assignment

Напиши програму, яка виводить цифри від `0` до `9` (у порядку зростання) та перехід на новий рядок.

### Expected files

- `ft_print_numbers.c`

### Allowed functions

- `write`

### Example

```
$> ./ft_print_numbers
0123456789
$>
```
