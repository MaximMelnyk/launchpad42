# ft_putstr

## Assignment

Напиши функцію `ft_putstr`, яка виводить рядок (string) на стандартний вивід (stdout).

### Прототип

```c
void	ft_putstr(char *str);
```

### Expected files

- `ft_putstr.c`

### Allowed functions

- `write`

### Example

```c
ft_putstr("Hello World");
```

Output:

```
Hello World
```
