# fizz_buzz

## Assignment

Напиши програму, яка виводить числа від 1 до 100, кожне на новому рядку, з такими замінами:

- Для чисел, кратних 3, виводь `fizz` замість числа.
- Для чисел, кратних 5, виводь `buzz` замість числа.
- Для чисел, кратних і 3, і 5, виводь `fizzbuzz` замість числа.

### Expected files

- `fizz_buzz.c`

### Allowed functions

- `write`

### Example

```
$> ./fizz_buzz
1
2
fizz
4
buzz
fizz
7
8
fizz
buzz
11
fizz
13
14
fizzbuzz
[...]
```
