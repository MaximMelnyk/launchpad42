# Vim Drills

## Завдання

На 42 ти працюватимеш у Vim (або Emacs). Vim -- потужний текстовий редактор, який працює в терміналі. Спочатку він здається незвичним, але після практики стає надзвичайно ефективним.

Виконай 15 мікро-завдань нижче. Кожне тренує окрему навичку Vim.

### Підготовка

1. Відкрий термінал
2. Створи тестовий файл: `vim practice.txt`
3. Виконуй завдання по порядку

### Мікро-завдання

#### 1. Режими Vim
- Відкрий файл: `vim practice.txt`
- Натисни `i` для входу в **Insert mode**
- Напиши "Hello from Vim"
- Натисни `Esc` для повернення в **Normal mode**

#### 2. Збереження та вихід
- В Normal mode натисни `:w` + Enter (зберегти)
- Натисни `:q` + Enter (вийти)
- Або `:wq` (зберегти і вийти)
- Примусовий вихід без збереження: `:q!`

#### 3. Навігація
- Відкрий файл знову: `vim practice.txt`
- Додай 5 рядків тексту (Insert mode)
- В Normal mode: `h`(ліворуч), `j`(вниз), `k`(вгору), `l`(праворуч)
- `0` -- початок рядка, `$` -- кінець рядка
- `gg` -- початок файлу, `G` -- кінець файлу

#### 4. Видалення
- `x` -- видалити символ під курсором
- `dd` -- видалити цілий рядок
- `dw` -- видалити слово

#### 5. Скасування (Undo)
- `u` -- скасувати останню дію
- `Ctrl+r` -- повторити скасовану дію (redo)

#### 6. Копіювання та вставка
- `yy` -- скопіювати рядок (yank)
- `p` -- вставити після курсора (paste)
- `dd` потім `p` -- перемістити рядок

#### 7. Пошук
- `/слово` + Enter -- пошук вперед
- `?слово` + Enter -- пошук назад
- `n` -- наступне входження
- `N` -- попереднє входження

#### 8. Заміна
- `:s/old/new/` -- замінити перше входження в рядку
- `:s/old/new/g` -- замінити всі входження в рядку
- `:%s/old/new/g` -- замінити всі входження у файлі

#### 9. Візуальний режим
- `v` -- visual mode (виділення символів)
- `V` -- visual line mode (виділення рядків)
- Після виділення: `d` (видалити), `y` (скопіювати)

#### 10. Вставка в різних місцях
- `i` -- вставка перед курсором
- `a` -- вставка після курсора
- `o` -- новий рядок знизу
- `O` -- новий рядок зверху
- `A` -- вставка в кінці рядка
- `I` -- вставка на початку рядка

#### 11. Нумерація рядків
- `:set number` -- показати номери рядків
- `:set nonumber` -- приховати
- Перейти на рядок 5: `:5` або `5G`

#### 12. Табуляція та відступи
- В Normal mode: `>>` -- відступ вправо
- `<<` -- відступ вліво
- В Visual mode: виділи рядки, потім `>` або `<`

#### 13. Створення файлу з Vim
- `:e newfile.c` -- створити/відкрити новий файл
- Напиши простий C код
- `:wq` -- зберегти і вийти

#### 14. Кілька вікон
- `:split` -- горизонтальний поділ
- `:vsplit` -- вертикальний поділ
- `Ctrl+w, w` -- перемикання між вікнами
- `:q` -- закрити поточне вікно

#### 15. Налаштування для C
- Створи файл `~/.vimrc` з корисними налаштуваннями:

```vim
set number
set autoindent
set tabstop=4
set shiftwidth=4
syntax on
```

## Підказки

<details>
<summary>Підказка: найважливіше правило</summary>

Якщо ти загубився в Vim -- натисни `Esc` кілька разів, щоб повернутися в Normal mode. Потім можеш зберегти (`:w`) або вийти (`:q!`).

</details>

<details>
<summary>Підказка: інтерактивний тренажер</summary>

Запусти `vimtutor` в терміналі -- це вбудований підручник Vim, який крок за кроком навчає основам. Займає ~30 хвилин.

</details>
