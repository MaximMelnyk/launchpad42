# Phase 0 Gate Exam

## Правила

Це екзамен на перехід з Phase 0 до Phase 1. Він імітує формат справжнього examshell на Piscine 42.

### Формат

- **5 завдань** зростаючої складності
- **Обмеження часу:** 4 години
- **Прохідний бал:** 4 з 5 правильних
- **Спроби:** максимум 3 спроби з перервою 48 годин між ними
- **Часткове зарахування:** при 3/5 правильних отримуєш 60% XP та можеш спробувати ще раз

### Правила екзамену

- Працюй ТІЛЬКИ з `write()` -- жодного `printf`
- Жодного інтернету, жодних підказок, жодних нотаток
- Тільки `man` сторінки дозволені
- Компілюй з `-Wall -Wextra -Werror`
- Norminette обов'язкова
- Кожне завдання -- окрема папка

## Завдання 1: ft_putchar (Розминка)

**Складність:** 1/5 | **Час:** ~10 хв

Напиши функцію `ft_putchar`, яка виводить символ на стандартний вивід.

### Прототип

```c
void	ft_putchar(char c);
```

### Файл

`ex01/ft_putchar.c`

### Тест

```c
int	main(void)
{
	ft_putchar('4');
	ft_putchar('2');
	ft_putchar('\n');
	return (0);
}
```

Очікуваний вивід: `42`

## Завдання 3: ft_putnbr

**Складність:** 3/5 | **Час:** ~30 хв

Напиши функцію `ft_putnbr`, яка виводить ціле число на стандартний вивід.

### Прототип

```c
void	ft_putnbr(int nb);
```

### Файл

`ex03/ft_putnbr.c`

### Тест

```c
int	main(void)
{
	ft_putnbr(0);
	ft_putchar('\n');
	ft_putnbr(42);
	ft_putchar('\n');
	ft_putnbr(-42);
	ft_putchar('\n');
	ft_putnbr(2147483647);
	ft_putchar('\n');
	ft_putnbr(-2147483648);
	ft_putchar('\n');
	return (0);
}
```

Очікуваний вивід:
```
0
42
-42
2147483647
-2147483648
```

**Увага:** обробка `INT_MIN` обов'язкова!

## Завдання 5: ft_print_comb

**Складність:** 4/5 | **Час:** ~45 хв

Напиши функцію `ft_print_comb`, яка виводить усі різні комбінації трьох цифр у порядку зростання.

### Прототип

```c
void	ft_print_comb(void);
```

### Файл

`ex05/ft_print_comb.c`

### Правила

- Три цифри комбінації завжди в порядку зростання (наприклад, `012` -- так, `120` -- ні)
- Комбінації розділені `, ` (кома та пробіл)
- Остання комбінація -- `789` (без коми після неї)

### Очікуваний вивід (початок та кінець)

```
012, 013, 014, 015, 016, 017, 018, 019, 023, ..., 789
```

### Алгоритм

Три вкладені цикли `while`:
- Перша цифра: від 0 до 7
- Друга цифра: від (перша + 1) до 8
- Третя цифра: від (друга + 1) до 9
